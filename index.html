IntelLSTM

Run and train a (very stupid) ChatGPT-like AI directly in your broswer

Upload a model's JSON file with the file picker above and click 'Run Model' to run it, or click 'Train model' to generate one. You might also need to click 'Run Model' two times. I don't know why, but as long as it works, idgaf.

The model uses a LSTM architecture, and is trained off of question and answer pairs.
